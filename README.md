An interactive **Power BI Business Intelligence Dashboard** built using **MySQL, SQL, Power Query, and DAX** to analyze credit card transactions, customer behavior, and financial performance.

---

# 📌 Project Overview

This project transforms raw customer and transaction data into interactive dashboards that help analyze revenue trends, customer segments, payment methods, and overall business performance.

---

# 🎯 Business Objective

The dashboard helps answer key business questions:

* Which customers generate the highest revenue?
* Which card categories perform best?
* Which states contribute the most revenue?
* How do customer demographics affect revenue?
* How do revenue and transactions change over time?

# 💡 Key Insights

* Blue Card customers generated over **83%** of total revenue.
* Customers aged **40–50** contributed the highest revenue.
* Swipe transactions accounted for **63%** of total revenue.
* Graduate customers generated the largest revenue share.
* Businessman customers were the highest revenue-generating segment.
* Revenue steadily increased from **Q1 to Q4**.
* Texas, California, and New York were the top-performing states.

# 🔄 Project Workflow

1. Imported customer and transaction datasets into MySQL.
2. Cleaned and transformed data using SQL.
3. Connected MySQL with Power BI.
4. Performed ETL using Power Query.
5. Built a relational data model.
6. Created DAX measures and KPIs.
7. Designed interactive dashboards with slicers and navigation.

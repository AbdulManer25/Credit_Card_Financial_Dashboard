# 💳 Credit Card Financial Dashboard

An interactive Power BI dashboard project designed to analyze credit card transactions, customer behavior, revenue trends, and overall business performance using MySQL and Power BI.

## 📌 Project Overview

This project focuses on transforming raw credit card customer and transaction data into meaningful business insights through data cleaning, transformation, modeling, and interactive visualizations.

The dashboard helps businesses:

* Monitor revenue performance
* Analyze transaction trends
* Understand customer behavior
* Track key financial KPIs
* Support data-driven decision making

## 🚀 Features

✅ Revenue Analysis

✅ Transaction Performance Tracking

✅ Customer Segmentation

✅ Age Group Analysis

✅ Education Level Analysis

✅ Job Category Analysis

✅ State-wise Revenue Distribution

✅ Quarterly Revenue Trends

✅ Interactive KPI Dashboard

✅ Dynamic Filters & Slicers

## 🔄 Project Workflow

### 1️⃣ Data Import

* Imported raw customer and transaction datasets into MySQL.
* Created database tables and relationships.

### 2️⃣ Data Cleaning & Transformation

* Removed duplicate records.
* Handled missing values.
* Standardized data formats.
* Performed SQL-based data validation and transformation.

### 3️⃣ Database Integration

* Connected MySQL database with Power BI using MySQL Connector.
* Imported cleaned and transformed data.

### 4️⃣ Data Modeling

* Created relationships between customer and transaction tables.
* Optimized the model for reporting and analytics.

### 5️⃣ Dashboard Development

* Built interactive dashboards using Power BI.
* Created DAX measures and KPIs.
* Designed user-friendly visualizations and reports.

## 📊 Dashboard Insights

The dashboard provides insights related to:

* Total Revenue
* Total Transactions
* Total Customers
* Revenue by Age Group
* Revenue by Education Level
* Revenue by Customer Job
* Revenue by State
* Quarterly Revenue Trends
* Transaction Methods Analysis
* Customer Utilization Analysis

---

## 💡 Key Insights

* Total Revenue: 55.32M
* Total Transactions: 655.65K
* Total Customers: 10.11K
* Swipe transactions generated the highest revenue.
* Customers aged 40–50 contributed the largest revenue share.
* Graduate customers generated the highest revenue.
* Businessman category customers contributed the most revenue.
* Revenue peaked during Quarter 3.
